To-Do List Manager

About the Project :

A Java, Spring, and Thyme leaf Powered To-Do List Manager.

• • Implemented Spring Data JPA to efficiently manage the storage of To-Do lists, tasks, and user data within a relational database (MySQL). This optimization resulted in a 25% improvement in task management efficiency..

• • Executed critical CRUD functionalities, including GET, DELETE, POST, and UPDATE, enhancing user experience and streamlining task management processes..

• Designed to enhance productivity, the application empowers users to schedule tasks during their most focused and effective periods.p>

• Implemented a user-centric design resulting in a 25% increase in user satisfaction through features aligned with the preferences.

Technologies Used

            • Java
            • Spring boot
            • Thymeleaf for UI
            • Spring JPA
            • MySQL
            • IntelliJ Idea IDE.
